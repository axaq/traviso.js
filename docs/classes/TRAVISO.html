getEngineInstance

Defined in ../src/traviso/Traviso.js:70

Syntax

getEngineInstance

(
  • instanceConfig
  • [globalConfig]
)
EngineView static

Summary

Creates and returns an isometric engine instance with the provided configuration. Also initialises traviso global settings if it hasn't been already.

Parameters:

  • instanceConfig Object

    configuration object for the isometric instance

    • mapDataPath String

      the path to the xml file that defines map data, required

    • [assetsToLoad=null] Array(String) optional

      array of paths to the assets that are desired to be loaded by traviso, no need to use if assets are already loaded to PIXI cache, default null

    • [minScale=0.5] Number optional

      mimimum scale that the DisplayObjectContainer for the map can get, default 0.5

    • [maxScale=1.5] Number optional

      maximum scale that the DisplayObjectContainer for the map can get, default 1.5

    • [numberOfZoomLevels=5] Number optional

      used to calculate zoom increment, default 5

    • [initialZoomLevel=0] Number optional

      initial zoom level of the map, should be between -1 and 1, default 0

    • [instantCameraZoom=false] Number optional

      specifies wheather to zoom instantly or with a tween animation, default false

    • [tileHeight=74] Number optional

      height of a single isometric tile, default 74

    • [isoAngle=30] Number optional

      the angle between the topleft edge and the horizontal diagonal of a isometric quad, default 30

    • [initialPositionFrame] Object optional

      frame to position the engine, default { x : 0, y : 0, w : 800, h : 600 }

      • x Number
        x position of the engine, default 0
      • y Number
        y position of the engine, default 0
      • w Number
        width of the engine, default 800
      • h Number
        height of the engine, default 600
    • [pathFindingType=TRAVISO.pfAlgorithms.ASTAR_ORTHOGONAL] Number optional

      the type of path finding algorithm two use, default TRAVISO.pfAlgorithms.ASTAR_ORTHOGONAL

    • [followCharacter=true] Boolean optional

      defines if the camera will follow the current controllable or not, default true

    • [instantCameraRelocation=false] Boolean optional

      specifies wheather the camera moves instantly or with a tween animation to the target location, default false

    • [instantObjectRelocation=false] Boolean optional

      specifies wheather the map-objects will be moved to target location instantly or with an animation, default false

    • [highlightPath=true] Boolean optional

      highlight the path when the current controllable moves on the map, default true

    • [highlightTargetTile=true] Boolean optional

      highlight the target tile when the current controllable moves on the map, default true

    • [tileHighlightAnimated=true] Boolean optional

      animate the tile highlights, default true

    • [mapDraggable=true] Boolean optional

      enable dragging the map with touch-and-touchmove or mousedown-and-mousemove on the map, default true

    • [backgroundColor=null] Number(Hexadecimal) optional

      background color, if defined the engine will create a solid colored background for the map, default null

    • [useMask=false] Boolean optional

      creates a mask using the position frame defined by 'initialPositionFrame' property or the 'posFrame' parameter that is passed to 'repositionContent' method, default false

    • [callbackScope=null] Object optional

      the scope to apply when calling callback functions, default null

    • [engineInstanceReadyCallback=null] Function optional

      callback function that will be called once everything is loaded and engine instance is ready, needs 'callbackScope' property, default null

    • [tileSelectCallback=null] Function optional

      callback function that will be called when a tile is selected, needs 'callbackScope' property, default null

    • [objectSelectCallback=null] Function optional

      callback function that will be called when a tile with an interactive map-object on it is selected, needs 'callbackScope' property, default null

    • [objectReachedDestinationCallback=null] Function optional

      callback function that will be called when any moving object reaches its destination, needs 'callbackScope' property, default null

    • [otherObjectsOnTheNextTileCallback=null] Function optional

      callback function that will be called when any moving object is in move and there are other objects on the next tile, needs 'callbackScope' property, default null

  • [globalConfig] Object optional

    configuration object for the traviso engine

Returns:

EngineView:

a new instance of the isometric engine
